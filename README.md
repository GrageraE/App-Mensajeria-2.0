## Compilación
### Requisitos
1. Compilador de C++ que soporte Qt 6
2. CMake
3. Qt 6
4. El submódulo QtWebSockets
### Windows
Lo mejor es usar Qt Creator
### Linux
```
mkdir build & cd build
cmake ..
make
```
